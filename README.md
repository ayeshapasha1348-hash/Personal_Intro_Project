## 1. Project Overview

The Personal Introduction Program is a beginner-level Python project.
The goal of this project is to learn how to take user input and display a friendly, personalized message.

This program asks the user for basic personal details and then shows them in a welcome message.

---

## 2. Project Goals and Objectives

* To understand basic Python syntax
* To use the input() function
* To store user input in variables
* To display output using print() and f-strings
* To learn how to upload a project on GitHub

---

## 3. Setup Instructions

Follow these steps to run the project:

1. Install Python from [https://www.python.org](https://www.python.org)
2. Open Command Prompt or Terminal
3. Navigate to the project folder
4. Run the program using the command:
   python personal_intro.py
5. Answer the questions shown on the screen

---

## 4. Code Structure

The project has a simple and clear file structure:

* personal_intro.py : Contains the Python program
* README.md : Project documentation
* requirements.txt : No external libraries required
* screenshot.png : Screenshot of program output

All files are placed in one main project folder.

---

## 5. Program Description

The program asks the user the following questions:

* Name
* Age
* Favorite hobby
* Country
* Favorite food

After collecting the input, the program prints a friendly welcome message using f-strings.

---

## 6. Technical Details

* Language Used: Python
* Concepts Used:

  * input() function
  * Variables
  * print() function
  * f-strings
* No data structures or external libraries are used.
* The program runs in a simple sequential manner (line by line execution).
