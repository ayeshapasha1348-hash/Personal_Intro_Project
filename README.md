Personal Introduction Program

Project Overview

This is a beginner-level Python project.
The program asks the user for personal information and shows a friendly welcome message.

Project Objectives

- Learn basic Python programming
- Use input() to take user input
- Store values in variables
- Display output using print() and f-strings

Program Description

The program asks the following questions:

- Name
- Age
- Favorite hobby
- Country
- Favorite food

After taking input, it displays a personalized and friendly welcome message.

How to Run the Program

1. Install Python on your computer
2. Open Command Prompt or Terminal
3. Go to the project folder
4. Run this command:
python personal_intro.py
5. Answer the questions shown on the screen

Project Files

- personal_intro.py : Main Python program
- README.md : Project documentation
- requirements.txt : No external libraries required
- screenshot.png : Screenshot of program output

What I Learned

- How to take input from the user
- How to use variables in Python
- How to use f-strings for formatted output
- How to create and upload a project on GitHub

Conclusion

This project helped me understand the basics of Python and GitHub.
It is a good starting project for beginners.
